# Esercizio: Strutturazione Testo e Liste in HTML

Questo repository contiene un esempio pratico di come strutturare contenuti testuali e liste puntate utilizzando il linguaggio **HTML**. L'obiettivo è trasformare un semplice commento di testo in una struttura semantica corretta per il web.

---

## 📝 Descrizione dell'Esercizio

L'esercizio prende un set di istruzioni testuali ("Eat, Sleep, Repeat") e le organizza utilizzando i tag fondamentali per la formattazione.

| Tag | Nome | Funzione |
| :--- | :--- | :--- |
| **`<p>`** | Paragrafo | Organizza il testo in blocchi separati. |
| **`<ul>`** | Unordered List | Crea l'elenco puntato (il contenitore). |
| **`<li>`** | List Item | Definisce ogni singola voce dentro la lista. |
